HEDIS Compliance & Data Quality Dashboard

This project demonstrates my ability to analyze, validate, and visualize healthcare data for actionable population health insights. Using a synthetic dataset of 12,000 records—simulating real-world challenges like missing values, mismatched fields, and data errors—I built a comprehensive Tableau dashboard to support population health decision-making.

Key Features:

  • ✅ Chronic disease management compliance analysis (Diabetes, Hypertension, COPD, Heart Failure)
  • ✅ Preventive screenings (Breast Cancer, Colorectal Cancer, Childhood Immunizations, Flu Vaccine)
  • ✅ Benchmark comparison with West Virginia state averages
  • ✅ Data quality insights
  • ✅ High-risk ZIP code heatmaps for targeted outreach
  • ✅ Trends and disparities by condition, gender, race, and year

Tools Used: Tableau, Excel (Data Validation), SQL (Data Preparation)

Key Takeaways: Identified care gaps and compliance trends, validated healthcare data for accuracy, translated complex data into clear, actionable visualizations.
